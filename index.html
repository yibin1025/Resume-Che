    • 工業物聯網平台的主要功能是整合不同的通訊協定，包括通用的 Modbus RTU/UDP、Modbus TCP、RS232/485。 這些協定用於機台即時資料的蒐集，同時進行邊緣運算，以降低資訊中心的負擔。為了有效地讓其他系統利用這些資訊， 使用了MQTT的發佈/訂閱特性，建立了一個Proxy Server， 將不同通訊協定的資料蒐集、整合後發佈，以利其他系統透過訂閱方式取得機台相關的即時資訊。
    • 庫存管理系统主要是接手前任工程師留下的專案，不斷的進行優化，包括以下方面：
      • 使用 RedLock.net 搭配 redis 實現分散式鎖定，確保每次透過API存取資料庫時只有一個請求，避免高併發請求導致資料髒讀的問題。
      • 適時在資料表上加上索引，同時優化 LINQ/Lambda Expression，以縮短大量資料查詢的回應時間。
      • 定期將不常異動的資料更新至 memory 或 redis 中，減少系統頻繁從資料庫讀取大量資料，提升資料查詢的回應速度。
      • 從 ERP 讀取大量資料後，使用 bulk insert/update 將資料批量寫入資料庫，大符減少資料庫寫入/更新時間，避免因資料庫長時間被鎖定而導致系統其他功能失效。

    相關技術為 C#，.NET Core 3.1，Web API，Entity Framework Core，SQL Server，Kubernetes，Docker 及 Git。
